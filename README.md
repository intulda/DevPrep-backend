## DevPrep

CS 학습과 코딩 테스트 준비 과정을 구조적으로 관리하기 위한 개인용 백엔드 서비스 구현

### 배경

CS 및 코딩 테스트 학습 과정에서 학습 내용이 흩어지고, 복습 및 면접 대비가 체계적으로 이루어지지 않는 문제가 존재한다.

### 목표

* CS 학습 내용을 구조화하여 저장할 수 있다.
* 코딩 테스트 풀이 과정을 사고 단위로 기록할 수 있다.
* 학습 기록이 누적되며 면접 대비 자료로 활용이 가능해야 한다.

### 기능

1. CS 주제 등록 / 조회
2. CS 주제별 면접 질문 등록
3. CS 주제별 꼬리 질문 등록
4. 코딩 테스트 문제 기록
5. 문제별 접근 전략 기록
6. 문제별 시간복잡도 기록

### 규칙

* CS 주제는 개념 요약, 면접 질문, 꼬리 질문이 모두 입력되어야 저장 가능하다.
* 코딩 테스트 문제는 접근 전략과 시간복잡도가 입력되어야 저장 가능하다.

### 개념 정의 

- CS 주제와 면접 질문의 관계: \
  CS는 주제는 개념 단위 이고, 면접 질문은 CS에 종속 된다.
- 면접 질문과 꼬리 질문의 관계: \
  꼬리 질문은 면접 질문 속 심화 질문이라 다른 개념이다.
- 코딩 테스트 문제와 CS 주제의 관계: \
  코딩 테스트 문제는 하나 이상의 CS 주제와 연결 될 수 있다.
