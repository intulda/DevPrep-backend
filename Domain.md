## DevPrep

### DevPrep 도메인

CS 학습과 코딩 테스트 준비 과정을 구조적으로 기록하고, 학습 이력을 일관된 형태로 관리하기 위해 정의된 도메인이다. 

1. CS주제는 여러 개의 면접 질문 가질 수 있다. 
2. 하나의 면접 질문은 여러 개의 여러 꼬리 질문을 가질 수 있다. 
3. 하나의 코딩 테스트 문제는 여러 개의 CS주제와 연관 될 수 있다.

### 도메인 모델 

`CsTopic`: CS 주제

- CS 개념을 대표하는 도메인
- 하나 이상의 면접 질문을 포함한다

`InterviewQuestion`: 면접 질문

- 특정 CS에 대한 면접 질문
- 하나 이상의 꼬리질문을 가질 수 있다

`FollowUpQuestion`: 꼬리 질문

- 면접 질문을 심화하기 위한 추가 질문

`CodingProblem`: 코딩 테스트 문제

- 코딩 테스트 문제를 표현하는 도메인
- 하나 이상의 CS 주제와 연관 될 수 있다
